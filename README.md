# 智能Bi项目
## 项目介绍：基于Spring Boot+MQ+AlGC的智能数据分析平台。区别于传统Bl，用户只需要导入原始数据集、并输入分析诉求，就能自动生成可视化图表及分析结论，提高数据的分析效率。

### 项目代码：在branch分支

## 项目架构

### 前端

基于uniapp的框架进行开发

### 后端

Java Spring Boot (万用后端模板)

MySQL数据库

MyBatis-Plus 及 MyBatis×自动生成

Redis + Redisson限流

RabbitMQ消息队列

第三方Al SDK (AI能力)

JDK线程池及异步化
